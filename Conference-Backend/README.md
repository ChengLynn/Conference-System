# 海运圈会议管理系统 - 后端API (TypeScript + Express)

基于TypeScript和Express构建的现代化会议管理系统后端API。

## ✨ 特性

- ✅ TypeScript支持 - 类型安全，更好的开发体验
- ✅ RESTful API设计 - 规范的接口设计
- ✅ JWT认证 - 安全的用户认证机制
- ✅ 文件上传 - 支持图片上传和管理
- ✅ 数据验证 - 完整的数据验证层
- ✅ 错误处理 - 统一的错误处理机制
- ✅ 日志系统 - 完善的日志记录
- ✅ 测试覆盖 - 单元测试和集成测试
- ✅ Docker支持 - 容器化部署
- ✅ API文档 - 自动生成API文档

- MongoDB >= 6.0
- npm >= 9.0.0
